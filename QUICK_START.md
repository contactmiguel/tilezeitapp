# Quick Start Guide

## 1️⃣ Launch the App

```bash
cd "c:\Users\conta\OneDrive\Miguel Stuff\My Claude Code\Bisaware sft app"
npm run dev
```

Open: **http://localhost:3000**

---

## 2️⃣ Upload a Plan

1. Click **Upload File** button
2. Select a PNG, JPG, or PDF file
3. Your plan appears in the canvas

---

## 3️⃣ Switch Modes

Click the mode buttons in the top bar:

| Mode | Purpose |
|------|---------|
| **Select** | View project summary |
| **Scale** | Set scale (coming soon) |
| **Draw** | Mark zones (coming soon) |

---

## 4️⃣ Replace a Plan

- Click the **Replace** button above the canvas
- Upload a new file

---
