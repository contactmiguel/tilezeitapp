# Tile / Stone Takeoff App UI Shell

This kit contains the first implementation package for a Next.js / React UI shell.

## Contents

- Claude Code implementation instructions
- React components
- TypeScript types
- CSS workspace styling

## How to use

1. Create or open a Next.js App Router project.
2. Copy the `src` folder into the project.
3. Make sure the `@/*` path alias is configured.
4. Run the project.
5. Confirm the upload shell and mode switching work.

## Main user flow included

- Open workspace
- Switch between Select, Scale, Draw
- Upload PNG, JPG, or PDF
- Preview uploaded file
- Replace uploaded file
